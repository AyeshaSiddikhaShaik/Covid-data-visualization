# 📊 COVID-19 India Dashboard Analysis

An interactive Tableau dashboard that provides comprehensive insights into the COVID-19 pandemic across India through data visualization and analytics. The dashboard enables users to explore state-wise case trends, vaccination progress, testing statistics, mortality, and demographic distributions.

---

## 📌 Project Overview

This project demonstrates how Tableau can be used to transform raw COVID-19 data into meaningful business intelligence dashboards. The dashboard combines multiple visualizations to provide a clear understanding of India's pandemic situation.

---

## 🚀 Dashboard Features

- 🗺️ State-wise COVID-19 Death Analysis (Interactive Map)
- 📈 Confirmed, Recovered & Death Trends
- 💉 First vs Second Dose Vaccination Analysis
- 💊 Vaccine-wise Dose Administration
- 👥 Age Group Distribution
- 🚻 Gender Distribution
- 🧪 ICMR Testing Labs by State
- 📋 State-wise Testing Statistics
- 🔍 Interactive Dashboard Filters

## 📈 Key Insights

- Maharashtra recorded the highest number of COVID-19 deaths.
- Vaccination coverage increased significantly after the rollout of the first dose.
- Adults aged 20–39 represented the largest affected age group.
- Covishield accounted for the majority of administered vaccine doses.
- Testing infrastructure was concentrated in a few major states.
